Package: H2O32BitExcel

(replaces H2o32BitExcel7, H2o32BitExcel8 and H2o32BitBUSExcel8)

This package enables you to use functions for properties of water and steam inside Microsoft Excel Version '97 and '2000. Some function have the possibility to use the old IFC67 properties with a switch variable (optional argument).
Unit sytems available inside (mixing possible):

  • SI with [bar] and [°C]
  • British thermal units

For the units used please take the Excel function assistant. Argument and result unit is printed there.


Benefits of this package in comparison to the older ones are:

  • Using "xll" instead of "xla". This is much faster since the properties are now build-in functions instead of Visual Basic code.
  • No more annoying messages with missing links to the add-in when sharing xls files with colleagues. Excel does no longer add property file name to property function name when saving your files.
  • No more adding links to Excel Basic when using property functions in your own code. Just use it:
    value = Application.Run("hVon_p_t", press, temp)

Installation:

  1. If you have previous version of the Excel water steam properties installed, delete them from your hard disk. Open Excel. Excel will recognize the Add-In is no longer available. Go to the Add-In-Manager (Extras->Add-In Manager) and remove H2O32Bit* from the list. Important: Close Excel after this!
  2. Copy all the files from this package to a directoy you like.
  3. Open Excel again and open the Add-In-Manager. Press find and move to the directory you have copied the file to. Select H2o32BitExcel.xll and press OK to let Excel open the file. In the list of available packages you will find "H2o32BitExcel".
  4. Installation is now complete. Have a look at the function assistant.
    You will find 2 new categories with function within:
    "properties water steam SI" and
    "properties water steam BTU"
  5. If you have Excel (*.xls) files with water and steam property call from earlier packages:
    When you open such a file Excel will say that it is linked to files that are not available. Just delete the file and path name in front of the each water steam function used in your spreadsheet and save your file. This is a one time task and will never occur again with this file.

Available functions:

Functions names are build up from the symbols names.
Von can be translated as "as a function of"
Satt can be translated as "saturation state"
1s means saturated liquid state, 2s means saturated vapour state
Example: Temperature (t) as function of pressure (p) and enthalpy (h) tVon_p_h(pressure,enthalpy)


Functions marked with (##) have an optional boolean argument UseIF97. The default is true(using IAPWS-IF97). When you specify false the IFC67 calculation is taken.
